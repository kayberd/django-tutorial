### Create a virtualenv
```bash
python3 -m venv .venv
```

### Activate the virtualenv
```bash
source .venv/bin/activate
```
### Deactivate the virtualenv
```bash
deactivate
```

### Install django:
```bash
pip3 install django
```

### Start a project
```bash
django-admin startproject <project_name>
```

### Start an app
```bash
python manage.py startapp <app_name>
```
